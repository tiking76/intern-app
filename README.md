## Internship-project

## At First
You hava to create ```./config.js```.<br>
then, set your API host name and API key.
```javascript
const configInfo = {
  hostName: 'YOYR HOSTNAME',
  apiKey: 'YOUR API KEY',
}

module.exports = configInfo
```

## Build Setup
### npm

``` bash
# install dependencies
npm install

# serve with hot reload at localhost:3000
npm run dev
```

### yarn

#### if you don't have yarn.
```bash
npm install yarn
```

```bash
# install dependencies
yarn add

# serve with hot reload at localhost:3000
yarn start
```
